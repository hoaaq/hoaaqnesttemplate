## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# migrate
$ yarn migration:run

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
